Buddy Driller editable assets
=============================

Replace any of these PNG files to reskin the game without changing code.
Keep the file names the same so the game can find them.

Player sprites:
- player-down.png
- player-up.png
- player-left.png
- player-right.png
- player-dead.png

Block sprites:
- block-blue.png
- block-yellow.png
- block-green.png
- block-red.png
- block-purple.png
- block-hard.png
- air-capsule.png

Recommended size: 256 x 256 PNG with transparency for player sprites.
The renderer scales each image to the current board size.
